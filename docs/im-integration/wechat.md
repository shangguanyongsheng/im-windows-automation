# 微信集成指南

## 集成选项

微信集成有多种方式，根据需求选择:

### 1. 企业微信 (推荐)

适用于企业场景，功能完整，API 稳定。

**准备工作:**
- 创建企业微信应用或机器人
- 获取 CorpID、CorpSecret 和 AgentId
- 配置可信域名和回调 URL

**配置环境变量:**
```env
WECHAT_CORP_ID=your_corp_id
WECHAT_CORP_SECRET=your_corp_secret
WECHAT_AGENT_ID=your_agent_id
```

### 2. 微信公众号

适用于面向公众的服务。

**准备工作:**
- 注册服务号（订阅号功能有限）
- 配置服务器 URL 和 Token
- 获取 AppID 和 AppSecret

**配置环境变量:**
```env
WECHAT_APP_ID=your_app_id
WECHAT_APP_SECRET=your_app_secret
WECHAT_TOKEN=your_token
```

### 3. 个人微信 (第三方库)

通过第三方库如 itchat、wxpy 等实现，但存在被封号风险。

**注意**: 不建议在生产环境中使用个人微信自动化。

## 实现建议

推荐使用企业微信方案，因为:
- 官方支持，稳定性好
- 功能完整，支持丰富的消息类型
- 安全性高，有完善的权限控制
- 适合企业自动化场景

## 安全注意事项

- 使用 HTTPS 加密通信
- 验证微信服务器签名
- 不要在代码中硬编码敏感信息
- 定期轮换密钥
- 限制应用权限范围

## 测试

1. 在企业微信/公众号中添加应用
2. 向应用发送测试消息
3. 检查日志确认消息接收和处理正常

## 故障排除

- **签名验证失败**: 检查 Token 和签名算法
- **认证失败**: 检查凭证是否正确
- **无法接收消息**: 确认服务器 URL 可访问且配置正确
- **消息发送失败**: 检查用户是否有权限接收消息
